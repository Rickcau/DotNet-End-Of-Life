# The Evolution: .NET Framework -> Modern.NET
**What is Modern .NET?**
Modern .NET (now just called ".NET") is Microsoft's reimagined, cross-platform version of the .NET development platform. It started as ".NET Core" in 2016 and was rebranded to just ".NET" starting with version 5.0 in 2020.

## The Two Worlds of .NET## 
**Think of it like this:**
┌─────────────────────────────────────────────────────────────┐
│  THE OLD WORLD: .NET Framework (2002-2022)                  │
├─────────────────────────────────────────────────────────────┤
│  • Windows-only                                             │
│  • Deeply integrated with Windows OS                        │
│  • Monolithic (everything included, whether you need it)    │
│  • Slow release cycle (years between versions)              │
│  • Latest version: 4.8.1 (2022) - FINAL VERSION             │
│  • Still supported, but no new features                     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  THE NEW WORLD: Modern .NET (2016-present)                  │
├─────────────────────────────────────────────────────────────┤
│  • Cross-platform (Windows, Linux, macOS)                   │
│  • Modular (install only what you need)                     │
│  • Open source                                              │
│  • Fast release cycle (annual releases)                     │
│  • Cloud-optimized                                          │
│  • Container-friendly                                       │
│  • High performance focus                                   │
│  • Current versions: .NET 8, 9, 10                          │
└─────────────────────────────────────────────────────────────┘
